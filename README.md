# 🚀 CloudQueueX AI

AI-Powered Event-Driven Ticket Processing System on AWS

---

## 📌 Project Overview

CloudQueueX AI is a cloud-native support ticket processing platform built using AWS services and AI-assisted workflows.

The system uses asynchronous event-driven architecture to process support tickets efficiently and scalably.

---

## ☁️ AWS Services Used

- Amazon EC2
- Amazon SQS
- Amazon DynamoDB
- Amazon SNS
- AWS IAM
- AWS CloudFormation
- Application Load Balancer

---

## 🤖 AI Features

- AI-powered ticket analysis
- Intelligent classification
- Priority detection
- Fallback AI logic
