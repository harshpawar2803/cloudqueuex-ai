# CloudQueueX AI

CloudQueueX AI is an AWS cloud-native event-driven ticket processing platform.

## Features

- Ticket Submission Portal
- AI Ticket Classification
- Priority Detection
- Team Assignment
- Amazon SQS Queue Processing
- DynamoDB Storage
- SNS Notifications
- Dashboard
- Search Tickets
- Event Driven Architecture

## AWS Services

- EC2
- SQS
- DynamoDB
- SNS
- IAM
- CloudWatch
- CloudFormation
